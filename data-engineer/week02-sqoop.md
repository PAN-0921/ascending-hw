
## Solution 5
1. run with one map task
```bash
sqoop import \
-m 1 \
--connect jdbc:mysql://database.ascendingdc.com:3306/retail_db \
--username=student \
--password=1234abcd \
--compression-codec=snappy \
--as-avrodatafile \
--table order_items \
--target-dir /user/pan/retail_db/avro
```
2.run with two map tasks
```bash
sqoop import \
-m 2 \
--connect jdbc:mysql://database.ascendingdc.com:3306/retail_db \

--password=1234abcd \
```
